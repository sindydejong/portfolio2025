How to push code to pages:

1. in fork: push everything to main (geen commando's nodig als je fork gebruikt)
git checkout main
git pull origin main
git add .
git commit -m "Update content"
git push origin main

2. Nadat main goed staat. Van main -> live (dus gh-pages)
git checkout gh-pages
git merge main
# (los conflicten op als nodig)
git push origin gh-pages

3. in main

# 1. Bouw je project
npm run build

# 2. Deploy naar gh-pages branch
npm run deploy

# 1. Werk en push code naar main
git checkout main
git pull origin main
git add .
git commit -m "Update content"
git push origin main

# 2. Build project
npm run build

# 3. Deploy build naar gh-pages
npm run deploy
