How to push code to pages:

ALLES VANUIT MAIN.
npm run deploy gebruikt het gh-pages npm pakket om automatisch de inhoud van je build-map (dist/) te pushen naar de gh-pages branch.
Daardoor hoef je zelf niet met git te switchen of handmatig te mergen.

# 1. Werk en push code naar main (doe je via fork)
git checkout main
git pull origin main
git add .
git commit -m "Update content"
git push origin main

# 2a. Build project (via command line)
npm run build
# 2b. Deploy build naar gh-pages
npm run deploy
